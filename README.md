# Learning Java
Personal repository documenting my journey through Java Backend
Development and DSA principles.
## Package Structure
I have organized my learning into specific packages for better
maintainability:
- **`basics`**: Core syntax, primitives, and type casting.
- **`controlflow`**: Logic building, loops, and conditional
statements.
- **`oop`**: Object-Oriented Programming (Abstraction, Inheritance,
etc.).
- **`io`**: File handling and Serialization.
- **`collections`**: Implementation of Sets, Lists, and Maps.
## Development Setup
- **IDE**: IntelliJ IDEA
- **Version Control**: Git
